# Music Audio Analysis Project

This project analyzes music audio files, converts them to MIDI, detects mistakes, and validates results.

## Structure
- `src/` — Source code and main package (`music_detector/`)
- `input/` — Input audio files
- `output/` — Output and results
- `tests/` — Unit tests

## Setup
- Use `requirements.txt` or `environment.yml` to install dependencies.

## Usage
See scripts in `src/` for entry points.
